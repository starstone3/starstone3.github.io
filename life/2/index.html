    高级数据结构与算法分析(王灿)¶

    WC老师的班里很多ACM大佬,不过没基础的同学也不用害怕,老师讲课还是比较生动的,就是声音有一些催眠

    wc老师是为数不多保留了Project互评阶段的老师,我们当时有两个Project忘记互评了,平时分直接-2(悲),还好做了两个bonus,最后算算平时分扣了1分左右

    ADS前半学期讲数据结构,包括红黑树,AVL树,B+树等,后半学期讲算法,一般而言,期末考中算法题普遍比数据结构等题难. 我的课程笔记与资料都在这里

    ADS的有期中和期末考试,期中考试一般是从题库里抽,期末考试是原创题,很可怕,因为这门课难度很大,并且期末考有40分斩杀线.

    并且期末考做历年卷的意义也不大,可以刷刷保持手感

    难绷的是我们这一届的期末考有两道题的选项有问题,最后部分人还加分了.

    总之,ADS在平时一定要好好听课,如果选择期末补天的话,要小心自己不要被斩杀

    计算机组成(马德)¶

    马德yyds!讲课十分生动,喜欢抽后排的同学回答问题.

    • 有一次期中考,不过一般和往年的卷子有较多重合,可以去智云里翻一下.

    • md老师会点名,并且没到对于平时分的惩罚还是挺高的.不过一般是第一节课上课前说要点名,然后第一节课下课的时候点

    • 期末考风格还是不好把握的,可以抄一张手写A4

    计组有六个实验,分别是:

    1. Warm up,设计ALU,寄存器等,为后面的CPU做准备

    2. 完成CPU测试平台的搭建,也就是做一个工程,之后每次新设计完CPU,就把这个工程里的CPU核心替换掉并测试

    3. 浮点数等操作实现,纯仿真

    4. 单周期CPU实现,分为三部分来实现:

      • 简单单周期CPU,仅支持算数指令等

      • 指令拓展的单周期CPU,支持分支指令等

      • 支持指令中断的单周期CPU,去年作为bonus

    5. 流水线CPU设计,分为两部分:

      • 普通的五段流水线CPU设计

      • 增加了冒险检测与相应操作的五段流水线CPU设计

      • 冒险部分去年作为bonus

    6. Cache设计,纯仿真,去年作为bonus

    普通物理学II(方明虎)¶

    虎哥的普物II前半学期讲电磁学,后半学期讲光学与量子力学.虎哥本人是电磁学的大牛,所以会偏电磁学一点

    评分标准不明,主要任务是每周的作业,一次期中考,一次期末考,还有一次浙江省物理竞赛获奖可以加分,一等奖3分,二等奖2分,三等奖1分,直接加到总评.

    去年的期中考相当惨烈,两个班两百人大概只有30人左右及格.想要前面年份回忆卷的可以去智云里翻一下,虎哥每次都会有一节课讲期中卷子

    我自己做的粗浅的笔记

    期中卷子:

    普物期中

    去年由于虎哥后面去出高考物理卷了,所以光学和量子力学部分是由另一位老师来上的.

    对于期末考试:

    • 不要只以历年卷为准,前年的期末几乎没考电磁学,所以有人说虎哥期末不考期中前的内容,结果去年小题大题全是电磁学

    • 量子力学一定要背薛定谔方程,在一维定态等,每年量子力学几乎只考这一道

    • 光学不仅要会算,还有会背公式.去年填空直接让你默写磨镜者公式

    毛概H(傅夏仙)¶

    老师虽然在cls的评分不高,但一学期上下来还是很不错的.平时分分为个人部分与小组部分:

    • 个人部分(10%):

      • 点名加小测,共两次,会提前说

      • 智慧树上把所有的视频刷完

    • 小组部分(50%):

      • 两次讨论作业,就是根据主题找资料,然后在三节课的最后一节课作PPT汇报

      • 一次经典研读,文献自选,需要提交读书报告,同时作PPT汇报

      • 一次实地调研,可以选择五四宪法馆等地方,同样需要完成调研报告与PPT汇报

      • 另外,除了经典研读和实地调研汇报的课,每次课都有讨论题,可能你们组在这次不需要展示,但还是需要以组为单位提交一份小组讨论结果

    最后小组的分数是一个0-100的整数,代表所有人的平均分.fxx老师这方面的分数还是很慷慨的,好像最高97,最低94

    期末考试(40%),纯背,98资料充足,如:

    • https://www.cc98.org/topic/6181236

    数据库系统(孙建伶)¶

    无敌好老师!!可惜孙老师马上就要退休了,但孙老师和他的助教为数据库这门课做出了很大的贡献,包括minisql的框架,Lab5的框架等.

    数据库这门课内容较多,每周都会讲一章,但难度还好.我的笔记:

    • https://starstone3.github.io/incourse/db/

    分数占比:

    • 平时(50%):

      • 每周作业与小测

      • 五次普通实验,包括:

        • 实验1 DBMS的安装和使用

        • 实验2 SQL数据定义和操作

        • 实验3 SQL数据完整性

        • 实验4 SQL 安全性

        • 实验5 数据库程序设计,有一个前端bonus,做了平时分+1

      • minisql大程,建议是:

        • 趁早组队,找好认识的队友,如果是陌生人的话,在沟通和最后的统一上可能会有问题

        • 趁早开始.五个普通实验是在春学期做完的,整个夏学期的任务就是做完minisql,很多人会拖到最后两周去做,后果就是一出事情就会来不及.minisql是一个很大的工程,每部分的编写就有难度,每部分整合起来更会出现各种各样的问题.比如我们的在插入很多数据的时候就会报段错误,最后神奇地把框架里的memcpy改为memomove就好了.

        • 我们是在四月中下旬就开始了minisql,最终在五月底左右结束,个人感觉还是无痛的.

    • 期末考(50%),允许带一张A4,从我们这届开始题型改为全是选择,判断与填空,方便改卷.题量大大增加,貌似有60多道.复习资料有:

      • 历年卷:https://www.cc98.org/topic/6211740/1#3

      • 学长的A4:https://note.hobbitqia.cc/DB/

    计算机体系结构(王小航)¶

    强推王小航老师!虽然是新老师,但事少给分好.平时上课不会点名,实验验收也很轻松.

    体系这门课的内容包括:

    1. Introduction.介绍一下计算机体系架构

    2. Memory.与计组重合内容较大,最后考试的部分也与计组很相似,如果计组学的好的话半个学期不用听课.

    3. ILP.除了介绍计组中学过的流水线意外,还讲了Branch Prediction,tumasulo和scoreboard等乱序执行的算法.

    4. DLP.介绍Vector Processing与SIMD,以及GPU的相关内容.

    5. TLP.介绍多核处理器的相关内容,包括Cache Coherence等.

    实验共六个加一个bonus,借用一下图灵网的介绍:

    • lab1：forwarding + predict not taken 的流水线

    • lab2：带异常处理的流水线

    • lab3：二路组关联 + alloc + write back 策略的 cache

    • lab4：cache 加入流水线

    • lab5：简单乱序流水线

    • lab6:scoreboard设计

    • bonus:使用gem5等工具探究CPU各项设置对性能影响,比如核数,L1 Cache大小等.推荐做一下,因为不会耗费太多时间,并且王老师对于bonus的给分似乎挺慷慨的.

    我的笔记在这里

    对于期末复习,值得注意的是wxh老师讲的不是很全,建议还是看看cr老师的PPT.期末允许带一张A4

    相关资料:

    • cr PPT: slides.zip

    • 前人复习整理:https://tsuki0512.github.io/2024_fall/CA/final_review/

    • 98帖子:https://www.cc98.org/topic/6082405

    • 课程共享计划

    • 学长的资料:https://note.hobbitqia.cc/CA/

    面向对象程序设计(陈翔)¶

    cx老师,我心目中OOP这门课最好的老师.上课时有PPT,但主要是用vim编写示例代码,为我们展示C++的特性.

    OOP内容包括:

    • 类(定义,继承,虚函数等)

    • STL(容器,算法,迭代器等)

    • 模板(函数模板,类模板,模板特化等)

    • 异常处理

    • 文件操作等

    我的笔记在这里

    分数组成:

    • 平时50%,包括:

      • 七次作业,每次大概就几道选择题或者一道程序填空题

      • 七个Lab,工作量不大:

        • Students I,设计统计学生成绩信息的类

        • Calendar II,打印指定年份的日历,有格式要求

        • Adventure,命令行上的冒险游戏

        • Personal Diary,实现个人日记,有写日记,读取日记,查找日记等功能

        • Fraction,设计一个分数类,具备一定功能

        • Vector,设计STL中的vector容器,包括迭代器等,使用template来泛型化

        • Object Serialization,大作业,有的班是内存池.实现标准类型(整数,字符串等)、STL容器与自定义类型的序列化与反序列化,包括二进制和XML格式两种.
